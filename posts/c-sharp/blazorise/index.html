Blazor Charts using Blazorise

2020-12-07 :: andy
#c-sharp  #blazor 

Demo of Blazorise which is a component library for Blazor. One of the interesting components is charting, which is a blazor wrapper for Chart.js

This live demo shows client side blazor (wasm) generating charts
